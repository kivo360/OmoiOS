## API Integration

### Backend Endpoints

All cost management endpoints are prefixed with `/api/v1/costs/`.

---

### GET /api/v1/costs/records
**Description:** List cost records with optional filtering

**Query Params:**
- `task_id` (optional): Filter by task ID
- `agent_id` (optional): Filter by agent ID
- `limit` (default: 100, max: 1000): Maximum records to return

**Response (200):**
```json
[
  {
    "id": "uuid",
    "task_id": "task-uuid",
    "agent_id": "agent-uuid",
    "provider": "openai",
    "model": "gpt-4-turbo",
    "input_tokens": 4521,
    "output_tokens": 1203,
    "total_tokens": 5724,
    "cost_usd": 0.0824,
    "created_at": "2025-01-15T10:30:00Z"
  }
]
```

---

### GET /api/v1/costs/summary
**Description:** Get aggregated cost summary for a scope

**Query Params:**
- `scope_type` (required): `global`, `ticket`, `agent`, `phase`, or `task`
- `scope_id` (required unless global): ID of the scoped entity

**Response (200):**
```json
{
  "scope_type": "ticket",
  "scope_id": "ticket-uuid",
  "total_cost_usd": 45.20,
  "total_tokens": 1250000,
  "input_tokens": 980000,
  "output_tokens": 270000,
  "breakdown_by_provider": {
    "openai": {
      "cost_usd": 32.40,
      "tokens": 850000
    },
    "anthropic": {
      "cost_usd": 12.80,
      "tokens": 400000
    }
  },
  "breakdown_by_model": {
    "gpt-4-turbo": {
      "cost_usd": 32.40,
      "tokens": 850000
    },
    "claude-sonnet-4-20250514": {
      "cost_usd": 12.80,
      "tokens": 400000
    }
  },
  "period_start": "2025-01-01T00:00:00Z",
  "period_end": "2025-01-15T23:59:59Z"
}
```

### GET /api/v1/costs/budgets
**Description:** List all budgets

**Query Params:**
- `scope_type` (optional): Filter by scope type

**Response (200):**
```json
[
  {
    "id": "uuid",
    "scope_type": "global",
    "scope_id": null,
    "limit_usd": 500.00,
    "current_spend_usd": 340.20,
    "utilization_percent": 68.04,
    "alert_threshold_percent": 80,
    "is_hard_limit": true,
    "is_active": true,
    "created_at": "2025-01-01T00:00:00Z"
  }
]
```

---

### POST /api/v1/costs/budgets
**Description:** Create a new budget limit

**Request Body:**
```json
{
  "scope_type": "ticket",
  "scope_id": "ticket-uuid",
  "limit_usd": 50.00,
  "alert_threshold_percent": 80,
  "is_hard_limit": false
}
```

**Response (200):**
```json
{
  "id": "uuid",
  "scope_type": "ticket",
  "scope_id": "ticket-uuid",
  "limit_usd": 50.00,
  "current_spend_usd": 0.00,
  "utilization_percent": 0.00,
  "alert_threshold_percent": 80,
  "is_hard_limit": false,
  "is_active": true,
  "created_at": "2025-01-15T10:00:00Z"
}
```

---

### GET /api/v1/costs/budgets/check
**Description:** Check budget status for a scope

**Query Params:**
- `scope_type` (required): Scope type to check
- `scope_id` (optional): Scope ID

**Response (200):**
```json
{
  "scope_type": "ticket",
  "scope_id": "ticket-uuid",
  "has_budget": true,
  "limit_usd": 50.00,
  "current_spend_usd": 45.20,
  "utilization_percent": 90.4,
  "is_exceeded": false,
  "is_warning": true,
  "remaining_usd": 4.80
}
```

---

### POST /api/v1/costs/forecast
**Description:** Forecast costs for pending tasks

**Request Body:**
```json
{
  "avg_tokens_per_task": 5000,
  "provider": "openai",
  "model": "gpt-4-turbo",
  "safety_buffer_multiplier": 1.5
}
```

**Response (200):**
```json
{
  "pending_task_count": 15,
  "estimated_cost_low": 12.50,
  "estimated_cost_mid": 24.80,
  "estimated_cost_high": 45.20,
  "assumptions": {
    "avg_tokens_per_task": 5000,
    "provider": "openai",
    "model": "gpt-4-turbo",
    "expected_retry_rate": 0.15,
    "safety_buffer": 1.5
  }
}
```
